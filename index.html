This is a classic, standard challenge: FizzBuzz

FizzBuzz takes in a number and returns 'fizz' if the number is divisible by 3, 'buzz' if the number is divisible by 5, and 'fizzbuzz' if the number is divisible by 3 and 5. Otherwise it will return the number input.
